Self Driving Car Algorithm

Chosen Framework - Utilitarianism

  • Utilitarianism is a moral choice that has the greatest benefit for the majority of people
  • Reasoning
    • I find that it provides a clear path on any moral decision
    • Due to the algorithm deciding the lowest danger weighting, it will result in the highest safety for the majority

Rationale

  • Optimisations
    • By creating multiple decisions for road selection, the algorithm filters any extra data that would end up being redundant
    • I created the algorithm to store the path to a node that has multiple options, as a result this will save time and resources from having to calculate that path again as it can head straight there

  • Balance Between Distance and Safety
    • As the ethical framework I have chosen is utilitarianism; when it comes to safety, the algorithm decides on the safest route regardless of distance
    • The distance is considered after the ethics of the decision. It will only have a significant impact on calculation if there are multiple safety weightings that are the lowest
